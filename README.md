# CI/CD com Next.js

Projeto desenvolvido para demonstrar uma pipeline completa de CI/CD utilizando Next.js, GitHub Actions, ESLint, Jest e deploy automatizado na Vercel.

## Tecnologias

- Next.js
- React
- TypeScript
- ESLint
- Jest
- GitHub Actions
- Vercel

## Funcionalidades

- Integração contínua com GitHub Actions
- Testes automatizados com Jest
- Verificação de código com ESLint
- Deploy automático na Vercel
- Interface moderna com Next.js

## Como executar o projeto

Clone o repositório:

```bash
git clone https://github.com/Geovanydev00/ci-cd-next.git
```

Entre na pasta:

```bash
cd ci-cd-next
```

Instale as dependências:

```bash
npm install
```

Execute o projeto:

```bash
npm run dev
```

## Testes

Executar testes:

```bash
npm run test
```

Executar lint:

```bash
npm run lint
```
